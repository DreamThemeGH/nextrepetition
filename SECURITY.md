# Security Policy

## Supported versions

Security fixes are applied to the current development branch and the latest release line.

## Reporting a vulnerability

Please report security issues privately instead of opening a public issue.

Recommended contact options:

- Repository owner: DreamThemeGH
- GitHub Security Advisories for the repository
- The contact address listed in `appinfo/info.xml`

## What to include

- A clear description of the issue
- Steps to reproduce
- Expected and actual behavior
- Any affected files, endpoints, or data paths

## What not to include

- Private keys
- Access tokens
- User data
